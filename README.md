# Epaper service

Rest API for managing e-papers. 

## Installation

1. Clone repository.

    ```bash
    git clone https://github.com/kudr9tov/epaper-service.git
    ```

2. Run docker-compose file.

    ```bash
    docker-compose up -d
    ```

## Usage

Open API is described in the `api-documentation.json` file.
